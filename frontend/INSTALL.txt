# FRONTEND (Professional Version) - Windows Setup

1. Go to the frontend folder in PowerShell:
   cd GenAI-Career-Advisor-Frontend

2. Install dependencies:
   npm install

3. Start the frontend:
   npm run start

4. Open your browser at:
   http://localhost:5173

5. Make sure backend is running at http://localhost:8000
   Then upload sample_data/sample_resume.txt to test.
