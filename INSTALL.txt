# INSTALL INSTRUCTIONS (Windows)

1. Install Python 3.10+ from https://www.python.org/downloads/
   - During installation, check 'Add to PATH'.

2. Install Node.js 18+ from https://nodejs.org/en/download/

3. Open PowerShell and go to backend folder:
   cd GenAI-Career-Advisor\backend
   python -m venv venv
   venv\Scripts\Activate.ps1
   pip install --upgrade pip
   pip install fastapi uvicorn
   python app.py

   -> Backend runs at http://localhost:8000
   -> Test it at http://localhost:8000/docs

4. Open a new PowerShell window, go to frontend folder:
   cd GenAI-Career-Advisor\frontend
   npm install
   npm run start

   -> Frontend runs at http://localhost:5173

5. In the browser, open http://localhost:5173
   - Upload sample_data/sample_resume.txt
   - See detected skills & recommendations
